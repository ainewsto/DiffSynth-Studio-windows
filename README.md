# DiffSynth Studio

## 原作者项目地址


## Installation 安装环境

Create Python environment:

```
cd DiffSynth-Studio-windows
conda create -p ./python python=3.9
```


activate the Python environment:

```
conda activate DiffSynthStudio
```

## Run (in WebUI) 启动UI界面

```
python -m streamlit run Diffsynth_Studio.py
```
